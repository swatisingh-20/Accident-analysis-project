
# Accident Analysis Project

This project analyzes accident data using Python, Pandas, SQLite and Matplotlib.

Tools Used:
Python
Pandas
SQLite
Matplotlib
Excel

Analysis Done:
Total accidents count
State wise analysis
Severity analysis
Weather analysis
Vehicles involved analysis

Dataset:
Sample dataset used for learning purpose.

Conclusion:
This project helps to understand accident patterns and can be useful for improving road safety.
